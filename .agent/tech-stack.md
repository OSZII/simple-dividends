# Technology Stack

The project is built with modern, efficient technologies focused on performance and developer experience.

## Frontend
- **Framework**: [Svelte 5](https://svelte.dev/) (utilizing runes and the latest Svelte features).
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for utility-first styling.
- **UI Components**: [DaisyUI](https://daisyui.com/) for pre-styled, accessible tailwind-based components.

## Backend & Database
- **Framework**: SvelteKit.
- **ORM**: [Drizzle ORM](https://orm.drizzle.team/) for type-safe database interactions.
- **Database Connector**: Drizzle (configured for PostgreSQL as per README).

## Directory Structure
- `/src/lib/components`: Shared, reusable components.
- `/src/lib/server/db`: Database schema and connection logic.
- `/src/routes`: Application pages and API endpoints.
