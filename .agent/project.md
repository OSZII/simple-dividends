# Project Strategy: Simple Dividends

## Overview
Simple Dividends is a web application designed to help users track dividend payouts and research stocks. The primary goal is to provide a clean, intuitive dividend calendar and a powerful yet simple stock screener.

## Main Features
- **Dividend Calendar**: The centerpiece of the application, showing upcoming dividend dates in a calendar view.
- **Stock Screener**: A tool to look up stocks and view basic metrics.
- **Stock Statistics**: Detailed views for stocks including:
    - Price
    - Price-to-Earnings (PE) Ratio
    - Revenue
    - Profit
    - Other key financial metrics

## Roadmap
1. **Dividend Calendar**: Core functionality for the main page.
2. **Stock Search**: Ability to find specific tickers.
3. **Stock Metrics**: Displaying financial data for each stock.
4. **Enhanced Screener**: Filtering and sorting stocks based on metrics.
